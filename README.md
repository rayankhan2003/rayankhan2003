## About

I am a **full-stack web developer** from Pakistan, growing from strong frontend foundations into product-focused application development. My recent work spans **Next.js, TypeScript, React, Tailwind CSS, Python, Flask, Prisma, and PostgreSQL** - with an emphasis on building practical workflows rather than only static interfaces.

I enjoy turning an idea into a clear, responsive experience, then connecting it to the APIs, database, authentication, document, QR, email, or data-handling logic it needs to become useful.

**Open to:** junior frontend or full-stack roles, practical collaborations, and projects where I can ship useful features and keep learning fast.
